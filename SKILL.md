# OriginPro 图表创建技能

创建出版质量的科学图表，支持25种论文常见图表类型。

## 功能概述

本技能提供完整的图表创建流程：
1. 数据导入和处理（CSV/Excel）
2. 智能图表类型推荐
3. 图表创建和样式定制
4. 导出为出版质量格式

## 用户引导流程

当用户只说"根据数据生成图表"或类似表述时，按以下流程引导：

### 第一步：获取数据信息

询问用户或读取数据文件，获取以下信息：
- 数据文件路径（CSV/Excel）
- 数据列名和数据类型
- 数据行数
- 是否有缺失值

### 第二步：分析数据特征

根据数据特征进行分类：

| 数据特征 | 说明 | 示例 |
|---------|------|------|
| **时间序列** | 有时间/顺序变量 | 日期、时间点、序号 |
| **分类数据** | 有类别变量 | 组别、类型、性别 |
| **连续数据** | 数值型变量 | 浓度、温度、评分 |
| **多变量** | 3个以上变量 | 多指标测量 |
| **配对数据** | 同一对象多次测量 | 方法比较、前后对比 |

### 第三步：推荐图表类型

根据数据特征推荐合适的图表，并说明优缺点：

#### 场景1：展示趋势变化
**数据特征**：有时间/顺序变量 + 连续变量

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **折线图** | 清晰展示趋势 | 不适合分类数据 | 时间序列、连续变化 |
| **面积图** | 强调累积效果 | 可能遮挡数据 | 累积变化、堆叠趋势 |
| **多线图** | 比较多条趋势 | 线条过多时混乱 | 多组时间序列 |

**示例问题**：
- "我想看数据随时间的变化"
- "展示实验结果的趋势"

#### 场景2：比较分类数据
**数据特征**：有分类变量 + 连续变量

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **柱状图** | 直观比较大小 | 不展示分布 | 组间比较 |
| **条形图** | 适合长标签 | 占用空间大 | 水平比较 |
| **分组柱状图** | 多系列并列 | 过于拥挤 | 多指标比较 |
| **堆叠柱状图** | 展示组成 | 难以比较中间值 | 比例比较 |
| **箱线图** | 展示分布 | 不够直观 | 分布比较 |
| **小提琴图** | 展示密度形状 | 解释成本高 | 分布形状比较 |

**示例问题**：
- "比较A、B、C三组的差异"
- "展示不同处理的效果"

#### 场景3：分析相关性
**数据特征**：两个连续变量

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **散点图** | 直观展示关系 | 只能展示2变量 | 两变量相关性 |
| **散点矩阵** | 多变量关系 | 图表较小 | 探索性分析 |
| **气泡图** | 三变量关系 | 大小难以比较 | 三变量分析 |

**示例问题**：
- "分析身高和体重的关系"
- "看看这两个指标是否相关"

#### 场景4：展示数据分布
**数据特征**：单个连续变量

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **直方图** | 展示频率分布 | 分箱影响结果 | 数据分布 |
| **箱线图** | 展示统计量 | 不展示形状 | 描述性统计 |
| **小提琴图** | 展示密度形状 | 解释成本高 | 分布形状 |

**示例问题**：
- "看看数据的分布情况"
- "检查是否正态分布"

#### 场景5：展示比例关系
**数据特征**：分类变量 + 数值（总和有意义）

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **饼图** | 直观展示比例 | 类别过多时混乱 | 简单比例 |
| **堆叠柱状图** | 比较多组比例 | 难以比较中间值 | 多组比例 |

**示例问题**：
- "展示各部分占总体的比例"
- "看看组成的分布"

#### 场景6：展示不确定性
**数据特征**：有均值和标准差/标准误/置信区间

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **误差棒图** | 展示不确定性 | 只展示误差范围 | Mean±SD/SE |
| **森林图** | 展示置信区间 | 专业性强 | Meta分析 |

**示例问题**：
- "展示实验结果的误差范围"
- "Mean±SD怎么画"

#### 场景7：多维数据比较
**数据特征**：多个指标

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **雷达图** | 多维对比 | 维度过多时混乱 | 能力评估 |
| **热力图** | 展示矩阵 | 颜色解释 | 相关矩阵 |

**示例问题**：
- "比较多个人在多个方面的能力"
- "展示相关性矩阵"

#### 场景8：特殊分析
**数据特征**：特定分析需求

| 推荐图表 | 优点 | 缺点 | 适用情况 |
|---------|------|------|---------|
| **生存曲线** | 展示生存率 | 需要删失数据 | 临床研究 |
| **Bland-Altman图** | 方法一致性 | 专业性强 | 方法比较 |
| **森林图** | Meta分析 | 需要效应量 | 系统综述 |

## 使用流程

### 步骤1：数据准备
```python
# 从CSV导入数据
from data_processing import load_csv, handle_missing_values, DataPipeline

df = load_csv('data.csv')
pipeline = DataPipeline()
pipeline.add_step(handle_missing_values, strategy='mean')
clean_data = pipeline.execute(df)
```

### 步骤2：选择图表类型
根据数据特征推荐图表类型：
- **时间序列数据** → 折线图、面积图
- **分类比较** → 柱状图、条形图、分组柱状图
- **相关性分析** → 散点图、散点矩阵
- **数据分布** → 直方图、箱线图、小提琴图
- **比例关系** → 饼图、堆叠柱状图
- **多维比较** → 雷达图、气泡图
- **统计分析** → 误差棒图、森林图、生存曲线

### 步骤3：创建图表
```python
import origin_graph_library as ogl

# 创建折线图
ogl.line_chart(x_data, y_data, title="图表标题", xlabel="X轴", ylabel="Y轴")

# 创建分组柱状图
ogl.grouped_column_chart(categories, data_dict, title="分组比较")
```

### 步骤4：样式定制
```python
# 设置坐标轴
ogl.set_axis_properties(gp, x_title="X Label", y_title="Y Label")

# 设置颜色
ogl.set_plot_color(gp, color='red')

# 添加图例
ogl.add_legend(gp)
```

### 步骤5：导出图表
```python
# 导出为PNG
gp.save_fig('output.png', dpi=300)

# 导出为SVG（矢量图）
gp.save_fig('output.svg')

# 批量导出
ogl.export_all_graphs(output_dir='output/', format='png', dpi=300)
```

## Nature级别样式指南

### 颜色方案
- 使用低饱和度颜色（NMI pastel系列）
- 避免过于鲜艳的颜色
- 使用颜色区分不同组别

### 字体规范
- 字体：Arial 或 Helvetica
- 大小：7-10pt（正文）、12-14pt（标题）
- 使用可编辑文本（SVG.fonttype = 'none'）

### 布局原则
- 白色背景
- 清晰的坐标轴标签
- 避免冗余元素
- 直接标签优于图例

### 导出标准
- 分辨率：600 DPI
- 格式：SVG、PDF、TIFF
- 可编辑文本

## 示例：创建论文图表

### 示例1：临床试验折线图
```python
from data_processing import load_csv, DataPipeline, handle_missing_values
import origin_graph_library as ogl

# 加载数据
df = load_csv('clinical_data.csv')

# 数据处理
pipeline = DataPipeline()
pipeline.add_step(handle_missing_values, strategy='mean')
clean_data = pipeline.execute(df)

# 按组分组计算均值和标准误
grouped = clean_data.groupby(['time', 'treatment'])['value'].agg(['mean', 'sem']).reset_index()

# 创建带误差棒的折线图
for treatment in grouped['treatment'].unique():
    data = grouped[grouped['treatment'] == treatment]
    ogl.error_bar_chart(
        data['time'], data['mean'], data['sem'],
        title="临床试验结果", xlabel="时间（周）", ylabel="指标值"
    )
```

### 示例2：分组柱状图比较
```python
# 准备数据
categories = ['Group A', 'Group B', 'Group C']

    categories, data_dict,
    title="组间比较", xlabel="组别", ylabel="测量值"

```

## 常见问题

### Q: 如何选择合适的图表类型？
A: 根据数据类型和展示目的选择：
- 趋势变化 → 折线图
- 分类比较 → 柱状图
- 相关性 → 散点图
- 分布 → 箱线图/小提琴图
- 比例 → 饼图

### Q: 如何达到Nature级别质量？
A: 遵循以下原则：
1. 使用专业配色方案
2. 保持简洁清晰
3. 使用可编辑文本格式
4. 600 DPI分辨率

### Q: 如何批量处理多个图表？
A: 使用`export_all_graphs()`函数或创建循环处理多个数据集。
